Projet Noka College

Ce projet est une application web full-stack développée avec React (frontend) et Node.js / Express.js (backend). Il permet de gérer les données des étudiants via des fichiers CSV et offre une interface utilisateur interactive.

Structure du projet

- frontend/ — Application React gérant l’interface utilisateur.  
- backend/ — Serveur Node.js Express qui traite les requêtes API et la manipulation des fichiers CSV.

Installation & Configuration

Backend

1. Aller dans le dossier backend :  
   bash
   cd backend
     
2. Installer les dépendances :  
   bash
   npm install
     
3. Démarrer le serveur backend :  
   bash
   npm start
   

Frontend

1. Aller dans le dossier frontend :  
   bash
   cd frontend
     
2. Installer les dépendances :  
   bash
   npm install
     
3. Démarrer le serveur de développement React :  
   bash
   npm start
   

Fonctionnalités

- Gestion des données étudiantes via import/export de fichiers CSV.  
- Interface utilisateur interactive développée avec React.  
- Backend RESTful avec Express.js.
